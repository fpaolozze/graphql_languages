# graphql_languages


## Node.js

```sh
$ yarn install
$ node src/index.js
```

## Go
get [graph-gophers/graphql-go](https://github.com/graph-gophers/graphql-go)

```sh
go run server.go
```

## Elixir
```sh
mix deps.get
mix phx.server
```

## Python (3.6)
```sh
python -m venv dev_env
pip install -r requirements.txt
python server.py
```
